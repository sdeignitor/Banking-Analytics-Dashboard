# 🏦 Banking Analytics Dashboard

## 📌 Project Overview

This project is an end-to-end Banking Analytics Dashboard built using MySQL and Power BI. It analyzes banking operations such as customers, accounts, loans, merchants, and transactions to provide business insights through an interactive dashboard.

The project demonstrates SQL data management, relational database design, data modeling, and business intelligence reporting.

---

## 🛠 Tools & Technologies

- MySQL
- SQL
- Power BI
- GitHub

---

## 📂 Dataset

The project consists of the following tables:

- Customers
- Accounts
- Transactions
- Loans
- Cards
- Merchants
- Branches

Dataset Size:

- 50,000 Customers
- 75,000 Accounts
- 30,000 Loans
- 100,000 Cards
- 2,000 Merchants
- 500 Branches
- 1,000,000 Transactions

---

## 📊 Dashboard Features

- Executive KPI Cards
- Customer Analysis
- Transaction Trend Analysis
- Account Distribution
- Customer Distribution by City
- Top Cities by Transaction Value
- Interactive Filters (City & Year)
- Drill-through Report Pages
- Dynamic KPI Measures

---

## 💡 Business Insights

- Banking transactions exceed several billion USD in total value.
- Customer distribution varies across cities.
- Account types are evenly distributed across the customer base.
- Interactive filtering enables city-wise and year-wise analysis.
- The dashboard provides an executive overview of banking operations.
